CDN

You can hotlink directly to the Transparensee CDN.

  • http://cdn.transparensee.com/simplicity/3.1/simplicity.min.js
  • http://cdn.transparensee.com/simplicity/3.1/simplicity.min.css

Older releases are available by changing the version number in the URL.

Bundled

You can bundle the JS and CSS with your application.

The prebuilt zipfile is available from the downloads page of the github project.

  • https://github.com/downloads/t11e/simplicity/simplicity-3.1.zip

Source

You can fork the project or build a custom release of the JS/CSS with just the components that you need.

  • http://github.com/t11e/simplicity
